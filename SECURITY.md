# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this project, please report it privately by emailing:  
**support@a-1apsvc.com**

We appreciate responsible disclosure and will respond promptly.  
Please include as much detail as possible to help us reproduce and address the issue.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| Older   | ❌        |
